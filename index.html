Frequently Asked Questions

What happens if I update versions?

Aura API Keys are universal and forward-compatible. A key generated today will work on all future V(x) versions seamlessly.

How do the 1000 Daily Credits work?

Each API key receives 1000 credits that auto-reset daily. Text costs 5c, Images 25c, and Video 50c per generation.

I cleared my browser cache and lost keys?

Go to 'Global Configuration' and use the Data Management tool to backup your keys as a JSON file, which you can restore anytime.
